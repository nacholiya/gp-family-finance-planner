Design Notes
The header toolbar uses visual icons instead of text: stacked family avatars for the member filter, a Singapore flag for language, a currency badge with dropdown, an eye icon for privacy mask toggle (green dot = visible), and a bell icon for notifications. The net worth chart has a pill-style time period selector (1W/1M/3M/1Y/All) with the active state using Heritage Orange. Three equal-width summary cards for Income (green), Expenses (orange), and Cash Flow (slate gradient) sit below the chart. "Upcoming Transactions" replaces recent activity to show forward-looking scheduled payments. All icons are SVG-based for crispness.
